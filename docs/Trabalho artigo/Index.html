Introducao

O HTML, ou Linguagem de Marcação de Hipertexto, é a pedra angular da internet moderna. Ele serve como a linguagem fundamental para estruturar e organizar o conteúdo das páginas da web. Imagine o HTML como o esqueleto de uma página web, fornecendo a estrutura que permite aos navegadores entenderem como apresentar informações, como texto, imagens e links. Cada elemento HTML age como um bloco de construção, e combinando-os de maneiras criativas, os desenvolvedores podem criar páginas web impressionantes e funcionais.

Enquanto o HTML dá forma e estrutura ao conteúdo de uma página web, o CSS, ou Folhas de Estilo em Cascata, é a ferramenta que confere estilo e beleza a essa estrutura. Com o CSS, os desenvolvedores têm o poder de controlar a aparência visual de cada elemento HTML. Isso inclui a escolha de cores, fontes, margens, espaçamentos e layouts. Imagine o CSS como o designer de interiores da web, decidindo como os elementos devem ser dispostos e estilizados para criar uma experiência de usuário atraente e harmoniosa. Juntos, o HTML e o CSS formam um duo poderoso que possibilita a criação das páginas web que navegamos todos os dias na internet.

A seguir uma breve explicação das principais tags.

Estrutura da Pagina HTML

Uma pagina HTML é composta por:

  • <!DOCTYPE html>: indica para o navegador utilizar a versão mais recente do HTML (HTML5)
  • <html lang="pt-br">: indica para o navegador que a pagina é HTML e a Lang indica o tipo de idioma a ser exibido
  • <Head>: e a tag que apresenta informações gerais sobre a página e de configuração
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: essa tag esta dentro da Head e é usada para informar o ajuste que a pagina apresentara no navegador.
  • <Title>: é a tag que informa o título de uma página na barra do navegador;
  • <Body>: e a tag que delimita o conteúdo da página
  • <Link>: é usada principalmente para conectar seu documento HTML a recursos externos, como folhas de estilo (CSS), ícones, arquivos de fonte e até mesmo feeds RSS. Ela fornece informações ao navegador sobre onde encontrar e como tratar esses recursos.

    • Uma informação importante a ser comentada é que a / dentro das tags é a delimitaçao que a tag esta fechada, porém nao são todas as tags que necessitam serem fechadas

    Tags de Texto

    Essas tags definem o conteúdo e a formatação de um documento HTML. Elas são usadas para estruturar o conteúdo de um documento, para formatar o texto, para adicionar imagens e outros elementos, etc...

  • <p>: essa tag define um paragrafo de um texto;
  • < h1/h2/...h6 > a tag H define um titulo, subtitulo, sendo que H1 é a mais importante, indo até H6 em grau de importancia.
  • <strong>: essa Tag funciona como o comando negrito que usamos nos editores de texto;
  • <cite>: aqui o texto ou palavra que é usado entre essa Tag fica em italico;
  • <Sup> / <Sub>: faz com que o trecho a que foi referenciado fique acima (sup) ou abaixo (sub) da linha;
  • <Strike>: faz com que o texto fique riscado;
  • <small> <big>: faz com que o texto fique menor(small) ou maior(maior);

    • Tags para Listas

    são usadas para criar listas de itens. Elas são essenciais para estruturar informações de forma organizada e lógica em uma página da web.

  • <ul>: cria uma lista não ordenada, ou seja, uma lista de itens que não estão em uma ordem específica;
  • <ol>: cria uma lista ordenada, ou seja, uma lista de itens que estão em uma ordem específica;
  • <li>: é usado dentro de uma lista para criar um item da lista.
  • <dl>: cria uma lista de definição, ou seja, uma lista de termos e suas definições;
  • <dt>: é usado dentro de uma lista de definição para criar um termo;
  • <dd>: é usado dentro de uma lista de definição para criar uma definição;

    • Tags de Links e Âncoras

    Essa classe de tag é usada para fazer referencias a documentos fora da pagina HTML

  • <a>: é a tag que cria um link, ou seja, um elemento clicável que leva o usuário a outro lugar;
  • <href>: é um atributo usado dentro da tag <a> para especificar o destino do link;
  • <target>: é um atributo usado dentro da tag <a> para especificar onde o destino do link deve ser aberto;
  • <rel>: é um atributo usado dentro da tag <a> para especificar a relação entre o documento atual e o destino do link;
  • <id>: é um atributo usado dentro da tag <a> para especificar um identificador exclusivo para um elemento HTML;
  • <img>: é a tag que adiciona uma imagem na pagina;

    • Tags de Formulários

    Essa classe de tag é usada para criar formulários, que são usados para coletar informações do usuário.

  • <form>: é a tag que cria um formulário;
  • <input>: é a tag que cria um campo de entrada de dados;
  • <tipe>: é um atributo usado dentro da tag <input> para especificar o tipo de campo de entrada de dados;
  • <name >: é um atributo usado dentro da tag <input> para especificar o nome do campo de entrada de dados;
  • <check-box >: é um atributo usado dentro da tag <input> para especificar que o campo de entrada de dados é uma caixa de seleção;
  • <button>: é a tag que cria um botão;
  • <label>: é a tag que cria um rótulo para um campo de entrada de dados;

    • Outras Tags Importantes

    Aqui é tags que nao consegui encaixar nos topicos anteriores mas são muito Importantes

  • <div>: é a tag que cria uma divisão ou seção na página;
  • <span>: é a tag que cria um contêiner para um pequeno trecho de texto;

    • Seletores em CSS

    Os seletores CSS são usados para "encontrar" (ou selecionar) os elementos HTML que você deseja estilizar.

  • Seletor Universal: * {propriedade: valor;} - seleciona todos os elementos HTML da página;
  • Seletor de Elemento: h1 {propriedade: valor;} - seleciona todos os elementos h1 da página;
  • Seletor de ID: #id {propriedade: valor;} - seleciona o elemento com o id especificado;
  • Seletor de Classe: .classe {propriedade: valor;} - seleciona todos os elementos com a classe especificada;
  • Seletor de Atributo: [atributo] {propriedade: valor;} - seleciona todos os elementos com o atributo especificado;

    • Aplicador de CSS

    Existem 3 formas possíveis de aplicar CSS a sua pagina, sao elas:

  • Por arquivos externos: é a forma mais comum de aplicar CSS a uma página. Para isso, você deve criar um arquivo CSS separado e vinculá-lo ao seu documento HTML usando a tag link;
  • Por tags internas: é uma forma de aplicar CSS a uma página sem a necessidade de criar um arquivo CSS separado. Para isso, você deve usar a tag style dentro da tag head;
  • Inline: é uma forma de aplicar CSS a um elemento específico. Para isso, você deve usar o atributo style dentro da tag do elemento;

    • Pricipais Comandos do CSS

  • background-color: define a cor de fundo de um elemento;
  • color: define a cor do texto de um elemento;
  • font-family: define a fonte do texto de um elemento;
  • font-size: define o tamanho do texto de um elemento;
  • font-weight: define o peso do texto de um elemento;
  • text-align: define o alinhamento do texto de um elemento;
  • width: define a largura de um elemento;
  • height: define a altura de um elemento;
  • margin: define a margem de um elemento;
  • padding: define o preenchimento de um elemento;
  • border: define a borda de um elemento;
  • border-radius: define o raio da borda de um elemento;
  • display: define o tipo de exibição de um elemento;
  • position: define o tipo de posicionamento de um elemento;
  • top: define a distância entre o elemento e o topo da página;
  • right: define a distância entre o elemento e a direita da página;
  • bottom: define a distância entre o elemento e o rodapé da página;
  • left: define a distância entre o elemento e a esquerda da página;
  • justify-content: define o alinhamento dos itens de um elemento;
  • hover: define o estilo de um elemento quando o mouse está sobre ele;
  • cursor: define o estilo do cursor do mouse quando ele está sobre um elemento;
  • font-display define como a fonte será exibida;

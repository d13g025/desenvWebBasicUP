Tags HTML Mais Comuns

Tag <h1> a <h6>

As tags de cabeçalho são usadas para definir títulos e subtítulos em uma página HTML. Elas variam de <h1> a <h6>, onde <h1> é o título mais expressivo e <h6> o menos expressivo.

Tag <p>

A tag <p> é usada para criar parágrafos de texto em seu documento HTML.

Tag <a>

A tag <a> é usada para criar links. O atributo href especifica o destino do link.

Tag <div>

A tag <div> é usada para criar divisões em seu documento, que podem ser estilizadas e manipuladas com CSS e JavaScript.

Tag <footer>

A tag <footer> é usada para definir um rodapé em uma seção ou página.
